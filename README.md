# Memory-Update-
New memory system 

Takes a script line by line and lets you memorize them with fuzzy logic so exact answers aren't necessary. The menu has these options:

Enter or 'n' for next, 
'p' for previous, 
's' to start over, 
'q' to quit, 
'h' to toggle help, 
'd' to display answer,
or enter an answer to try again.
